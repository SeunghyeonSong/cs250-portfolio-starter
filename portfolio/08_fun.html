Project Overview

This is my custom demo. Which will show my understanding about overall course. In this demo I try to use every knowledge I get from class.

Opengl Tech

In this paragraph I'm going to talk about opengl tech. Below list is mainly techinque I learned at class. (PS. I didn't used shadow mapping. Instead of I add new tech quite similar to it.)

  • I used procedural meshes techinque to make plane for sea and sphere for stars.
  • Then I used fog for making more natural view for player. I made it with shader.
  • I used toon shading based to make wired celstial body which looks like black hole + saturn. Also used shader.
  • I used value noise to made moon. Not likely star, moon doesn't need real time animtaion.
  • I used gradient noise to made sea texture and stars. This make objects more realistic.

I can't add new tech. I try to understand it but it has a lot of problem.

Fun Tech

In this paragraph I'm going to talk about other fun and simple tech except opengl tech.

  • To make sea and star surface look naturally. I made texture to move. And these made more natural.
  • To make black hole I used torus and this made more cooler.
  • And for making eclipse I just assume the situation when two objects start to collide. And when they collide I used easing to make it more natural.
  • To make zooming I used fov to be shrink with easing.
  • To make star field I used tech from zelda orcarina of time. To implement sky it used small sphere that keep follows camrea. And this was easy to implement. And for random star position I used hash. To make star not to cover other objects I disable depth test. And to set sky to be shown inside I change back face culling. How to play

Try to find 5 different Celestial body.

  • You can zoom the screen with mouse left button. It will also remove fog.
  • Right of the window there are imgui buttons. You can custom several objects.
