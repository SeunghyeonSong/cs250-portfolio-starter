Project Toon Shading

3D Graphics

Project Overview

Project checking knowledge of phong lighting and toon shading. In this project I try to find my weakness part of understanding.

Tasks Completed

  • Make normal matrix based on view view matrix and model matrix.
  • Make vertex shader which out normal vector
  • Make fragment shader which has toon shading.
  • Make variation of fragment shader which toon range is custom.
  • Make variation of fragment shader which toon range is uniform.
  • Make variation of fragment shader which toon range is gradient texture.
  • Make variation of fragment shader which toon range is custom.
  • Add fog shading at fragment shader for all fragement shader.
  • Finally do antialiasing for each fragment shader.
